Images
Images or responsive images are used for visual information displayed on websites that are screen responsive.
Squared & Round Images
Images with squared and rounded corners with specified default dimensions. Use class .img-square or .img-round to specify the image shape.
Responsive Image
Responsive Image is the one that expands to cover the container it is present in.
